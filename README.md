# DISCORD-C2 Server Documentation

## Overview

DISCORD-C2 is a command and control (C2) server for Discord. It allows you to control and manage your agents using Discord as a communication channel.

## Features

- Command execution
- File transfer
- Agent management
- Stealth mode
- ...
